## ЧТО Я ИЗУЧАЮ

### 1. ГЛАВНЫЙ ЭКРАН (PATH)

**Структура:**
- Горизонтальный путь (Path) с уроками
- Каждый урок = кружок/иконка
- Цветовая индикация:
  - 🟢 Зеленый = пройден (perfect)
  - 🟡 Желтый = нужно повторить (cracked)
  - ⚪ Серый = заблокирован
  - 🔵 Синий = текущий урок
- Соединительные линии между уроками
- Duo (совенок) появляется рядом с текущим уроком

**Элементы:**
- Header: XP, Hearts, Streak, Profile
- Path: горизонтальный скролл
- Кнопка "Начать урок" на текущем уроке

---

### 2. ЭКРАН УРОКА (LESSON SCREEN)

**Структура экрана:**

```
┌─────────────────────────────────────┐
│  [Duo]  ❤️❤️❤️❤️❤️  XP: 150  [Menu] │  ← Header
├─────────────────────────────────────┤
│  Прогресс: ████░░░ 4/7              │  ← Progress Bar
├─────────────────────────────────────┤
│                                     │
│         [Duo анимация]              │  ← Character (опционально)
│                                     │
│  Упражнение 4/7                     │  ← Exercise Title
│                                     │
│  "Hello" на русском:                │  ← Question
│                                     │
│  ┌─────────┐  ┌─────────┐         │
│  │ Привет  │  │ Здравствуй│        │  ← Options (2-4 варианта)
│  └─────────┘  └─────────┘         │
│  ┌─────────┐  ┌─────────┐         │
│  │ Пока   │  │ Спасибо  │         │
│  └─────────┘  └─────────┘         │
│                                     │
│  [Кнопка: Проверить]                │  ← Check Button (опционально)
└─────────────────────────────────────┘
```

**Ключевые элементы:**

1. **Header:**
   - Duo (совенок) - слева
   - Hearts (❤️❤️❤️❤️❤️) - 5 жизней
   - XP счетчик
   - Меню (справа)

2. **Progress Bar:**
   - Визуальный прогресс (1/7, 2/7...)
   - Процент заполнения
   - Анимация при переходе

3. **Exercise Area:**
   - Тип упражнения (Multiple Choice, Translation, etc.)
   - Вопрос/задание
   - Варианты ответов (2-4 кнопки)
   - Кнопка "Проверить" (иногда автоматически)

4. **Feedback:**
   - Мгновенная обратная связь после ответа
   - Зеленый ✅ = правильно
   - Красный ❌ = неправильно
   - Объяснение (появляется через 1-2 сек)
   - Автоматический переход к следующему (через 2-3 сек)

### 3. ТИПЫ УПРАЖНЕНИЙ (EXERCISE TYPES)

#### A. Multiple Choice (Выбор ответа)

**Визуально:**
- 2-4 большие кнопки
- Текст по центру кнопки
- При нажатии: мгновенная обратная связь
- Правильный ответ: зеленая рамка + ✅
- Неправильный: красная рамка + ❌

**Пример:**
```
Что такое личные границы?

[Правила родителей        ]
[Невидимый забор ✅       ]  ← Правильно
[Способ избегать людей    ]
[То, что делает эгоистом ]
```

#### B. Translation (Перевод)

**Визуально:**
- Фраза на одном языке
- Нужно выбрать перевод из вариантов
- Или ввести текст

**Пример:**
```
"Hello" на русском:

[Привет ✅]
[Пока]
[Спасибо]
[Пожалуйста]
```

#### C. Word Selection (Выбор слов)

**Визуально:**
- Фраза с пропусками
- Слова внизу для выбора
- Перетаскивание или клик

**Пример:**
```
Личные границы — это [забор], защищающий...

[забор] [стена] [правило] [закон]
```

#### D. Matching (Сопоставление)

**Визуально:**
- Два столбца
- Соедини пары
- Drag & drop или клик

#### E. Fill in the Blanks (Заполни пропуски)

**Визуально:**
- Текст с пропусками
- Выбор слова из списка
- Или ввод текста

---

### 4. МЕХАНИКА ОБРАТНОЙ СВЯЗИ

**После ответа:**

1. **Мгновенная визуальная обратная связь:**
   - Правильно: зеленая рамка, ✅, анимация
   - Неправильно: красная рамка, ❌, анимация

2. **Hearts система:**
   - Правильный ответ: Hearts не меняются
   - Неправильный: -1 Heart
   - Hearts = 0: урок прерывается (нужно восстановить)

3. **Объяснение:**
   - Появляется через 1-2 секунды
   - Текст объяснения
   - Почему ответ правильный/неправильный

4. **Автоматический переход:**
   - Через 2-3 секунды после feedback
   - Плавная анимация
   - Следующее упражнение

- Duo появляется рядом с текущим уроком

## ПРИМЕНЕНИЕ К НАШЕМУ ПРОЕКТУ

### АДАПТАЦИЯ ДЛЯ DA TEENS

**Вместо Duo → Катя (психолог):**
- Катя появляется в начале урока
- Комментирует прогресс
- Празднует успехи
- Поддерживает при ошибках

**Структура урока:**
- 5-7 коротких упражнений
- Только вопрос-ответ (без длинного текста)
- Мгновенная обратная связь
- Hearts система (5 жизней)

**Типы упражнений:**
1. Multiple Choice (из quiz)
2. Role-play (из practiceExercise)
3. Matching (из examples)
4. Fill in the Blanks (из ключевых фраз)

**Преобразование данных:**
- `lesson.quiz[0]` → Multiple Choice упражнение 1
- `lesson.quiz[1]` → Multiple Choice упражнение 2
- `lesson.practiceExercise` → Role-play упражнение
- `lesson.formats.text.content.examples` → Matching упражнение
